# FaceSharp Frontend

React приложение для FaceSharp.

## Установка

```bash
# Установить зависимости
npm install
```

## Запуск

```bash
# Запустить dev сервер
npm run dev
```

Приложение будет доступно на `http://localhost:3000`

## Сборка

```bash
# Собрать production версию
npm run build

# Предпросмотр production версии
npm run preview
```
